### Instructions de déploiement

1. **Cloner le repository:**
   ```bash
   git clone https://github.com/houssemeddinelassoued/ia-echotrack-labs-2026-tech+business.git
   ```

2. **Ajouter les fichiers au repository:**
   ```bash
   git add .
   git commit -m "Initial commit: Set up HTML project"
   git push origin main
   ```

3. **Activer GitHub Pages:**
   - Allez dans Settings du repository
   - Sélectionnez "Pages" dans le menu de gauche
   - Source: Branch `main`, dossier `/ (root)`
   - Sauvegardez
